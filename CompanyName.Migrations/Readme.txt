Утилита миграции.

	В процессе рефакторинга кода сайта на проекте возникает необходимость мигрировать данные из одной таблицы в другую совместно с переносом файлов хранящихся
на диске отдельно от SQL сервера или сконвертировать данные в определённый формат для всех пользователей. Такие миграции могут занимать продолжительное время и быть
прерваными посередине процесса выполнения. Использовать классические SQL скрипты для таких задач неудобно. Была написана небольшая утилита, предназначенная для
миграции любых данных с возможностью использования существующей инфраструктуры кода. 

	Данный пример кода содержит миграцию предназначенную для конвертации всех телефонных номеров присутсвующих в опросниках пользователей добавленных за всё время
существования системы к определённому формату. Их очень много.  Опросники пользователя содержащие телефонные номера храняться в соответсвии с концепцией EvenSourcing.
Утилита выполняет журналирование своих операций. Благодоря использованию многопоточности на продакшен базе процесс миграции занял всего 40 минут вместо 4 часов.
